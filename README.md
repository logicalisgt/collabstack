## 📖 About

**Collab Stack** is a deep technical blog focused on real-world design, deployment, and troubleshooting of Cisco Collaboration and Contact Center platforms.

This repository powers a GitHub Pages site that documents:

- ✅ Cisco CUBE (Call Routing, SIP, Dial Plans)
- ✅ Webex Calling Architecture & Integrations
- ✅ Webex Contact Center (Flows, Design, Debugging)
- ✅ End-to-End Call Flows & Signaling Analysis
- ✅ Operational Runbooks & Troubleshooting Procedures

---

## 🎯 Purpose

Most documentation covers *what* to configure —

✔️ **Why systems behave the way they do**

## 🗂️ Repository Structure

The project follows a standard GitHub Pages (Jekyll) layout:

    .
    ├── _config.yml        # Site configuration
    ├── index.md           # Homepage
    ├── about.md           # About page
    │
    ├── _posts/            # Blog posts (Markdown)
    │   ├── YYYY-MM-DD-title.md
    │
    ├── _layouts/          # Page templates
    │   ├── default.html
    │   ├── post.html
    │
    ├── _includes/         # Reusable components (header, footer)
    │
    ├── assets/            # Static files
    │   ├── css/
    │   ├── images/
    │   ├── diagrams/
    │
    └── README.md          # Repository documentation
